Using the UX principles we went over, you are going to build a shop for a small flower store.

  • Define your prime objective.
  • Make user stories for at least 3 users.
  • Create a full sitemap based on your user stories.
  • Create a low fidelity wireframe that includes all of the necessary elements according to the sitemap.

    • Prime Objective

  • Create a tool for a small flower shop to show off new seasonals, standard offerings, allow users to place orders, and check basic store details.

    • User Stories

  • Customer: Viewing homepage, select products, place orders
  • Store Employee: Viewing permissions + view access to order calendar, requests to drop/take orders
  • Manager: Previously mentioned viewing permissions + edit/approval access to order calendar and account management
